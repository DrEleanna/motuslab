- `script.js`
- `assets/brand/`
- `assets/banner/`

- Updated Credentials section:

- Improved overall typography with more refined fonts.

- Added complete local assets so the site can run as a standalone GitHub Pages site.

## Publish on GitHub Pages
Upload all contents to your repository root and then enable:

`Settings → Pages → Deploy from a branch → main → /root`
